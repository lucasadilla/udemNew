# Navbar images

Add these files for the header to display correctly:

- **logo-femmes-et-droit.png** – F&D / Comité Femmes et Droit UdeM logo
- **instagram.png** – Instagram icon

The navbar uses a black background with white text and icons. If your logo is already light/white, you may remove the `brightness-0 invert` classes from the header component on those images.
